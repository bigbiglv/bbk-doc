# 基本语法

## 标题
在文本前面加上不同数量的#号，最多到6个
```
# 标题
## 标题
...
######  标题
```

## 字体
* *斜体*：`*斜体*`
* **粗体**：`**粗体**`
* ***斜体加粗***：`***斜体加粗***`
* ~~删除线~~：`~~删除线~~`

## 分割线
使用三个或以上 `---` 或`***`
---

## 链接
- 网络链接 `[超链接名](网络地址 "超链接title")`
可不加title，跳转[简书](http://jianshu.com)

- 文章标题锚点 `[标题1](#heading-id)`
- 其他文章 `[跳转其他文章](./xxx.md)`


## 列表
```
* 有序列表：使用`序号.`后加空格
   1. 1
   2. 2
   3. 3
* 无序列表：使用 `*`，`+`, `-` 后加空格
* 列表嵌套：下级列表回车后敲三个空格
   * 下级列表
      * 下下级
```
