# 函数重载
同一个方法按照不同的参数返回不同的类型

## 函数重载
- 函数名相同
- 前面的同名函数作为类型选项, 最后一个函数为具体实现
```ts
const userList = [
  {
    name: 'xxx',
    age: 10
  }
]

function find(keyword: string): string
function find(keyword: number): number

function find(keyword: string | number) {
  if(typeof keyword === 'string') {
    return userList.find(user => user.name === keyword)?.name
  }
  if(typeof keyword === 'number') {
    return userList.find(user => user.age === keyword)?.age
  }
  return undefined
}
const res1 = find(1) // const res1: number
const res2 = find('1') // const res2: string
```

## 方法重载
- 和函数重载一样的道理, 作用于 `class` 中的方法
```ts
class Calculator {
  // 方法重载签名
  add(a: number, b: number): number
  add(a: string, b: string): string

  // 实现方法
  add(a: number | string, b: number | string) {
    if (typeof a === 'number' && typeof b === 'number') {
      return a + b 
    }
    if (typeof a === 'string' && typeof b === 'string') {
      return a + b 
    }
    throw new Error('Invalid arguments')
  }
}

const calc = new Calculator()
const numResult = calc.add(1, 2)       // const numResult: number
const strResult = calc.add('a', 'b')   // const strResult: string
```
