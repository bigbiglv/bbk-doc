# 进阶

## 性能优化

1. 前端性能从加载到渲染优化全链路方案
应该按客户端发起请求到页面渲染完毕的顺序来回答
1. 发起请求前
  1. 强缓存与协商缓存
  2. pwa的Service workers
2. 网络连接: 
  1. DNS 预解析
  2. TCP/TLS 预连接
  3. CDN 分发
3. 数据传输
  1. 包体积:  图片压缩 路由懒加载 组件懒加载 Tree Shaking
  2. 代码压缩: 代码打包压缩 和 服务器 Gzip 或 Brotli 资源压缩
4. 渲染
  1. 减少重绘和重排
  2. 长列表优化
